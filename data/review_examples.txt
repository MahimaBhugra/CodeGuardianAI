EXAMPLE 1

Code:

return a / b

Issue:
Possible division by zero.

Severity:
High

Fix:
Check if b equals zero before division.

EXAMPLE 2

Code:

query = "SELECT * FROM users WHERE id=" + user_input

Issue:
SQL Injection vulnerability.

Severity:
Critical

Fix:
Use parameterized queries.
